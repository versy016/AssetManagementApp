# Release — 2026-06-18 (pending)

_Changes since the previous release (commit `4d74ee6`)._

## Added
- **New Asset — address autocomplete:** Location fields now show Google Places
  address suggestions as you type (pick one to fill the field).
- **Swap QR — unified search:** A single "Search for an asset" box that matches
  on name, type, model, serial, asset ID, or assignee — shown above
  "Find by Asset ID" — replacing the separate Model / Type / Assigned fields.
- **End of Life — restore:** Admins now get a **"Bring back in service"** action
  when scanning an End-of-Life asset.

## Changed
- **Imported-asset quick actions:** Standardised the button grid — blue actions
  on the left, orange on the right — and "Assign Imported Asset" now stays on
  one line.
- **Places API:** The address-autocomplete endpoint now reports the real Google
  error (e.g. `REQUEST_DENIED`) instead of silently returning no results, so
  misconfiguration is visible.

## Fixed
- **iOS date picker:** Spinner text was washed out / unreadable — now
  high-contrast (forced light theme + explicit text colour).
- **New Asset (scan to select ID):** Form fields appeared ~1 second after
  scanning a QR; they now appear immediately.
- **QR-reserved assets:** The "Add image" button is hidden until a real asset is
  assigned to the QR.
- **Swap via Scan QR:** Scanning an empty / reserved / End-of-Life QR (or the
  same QR) is now blocked with a clear validation message instead of proceeding.
- **End-of-Life screen:** The "Back to Dashboard" button no longer truncates to
  "Back to …".
