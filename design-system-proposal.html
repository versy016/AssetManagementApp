Design System Proposal v1.0

GearOps Redesign

Modern Utility aesthetic — clean, functional, and premium. Designed for both field technicians scanning gear on-site and office managers tracking inventory at their desks.

Design Philosophy
⚡
Speed First
One-tap actions, large touch targets for gloved hands, minimal navigation depth. Field workers need to check in gear in under 5 seconds.
🛠
Practical Density
Information-rich without clutter. Status at a glance, smart defaults, and progressive disclosure. Show what matters, hide what doesn't.
🎨
Refined Polish
Subtle shadows, precise spacing, cohesive color language. Professional enough for client demos, tough enough for the job site.

Technical Implementation Plan
Styling
NativeWind v4
Tailwind CSS utility classes compiled for React Native. CSS variables map to a custom Tailwind config for consistent design tokens across Web, iOS, and Android.
Theme
Light Primary + Dark Toggle
React Context-based theme provider with system preference detection. All colors defined as CSS variables / Tailwind config tokens for instant theme switching.
Components
Composable Design System
Atomic components: Button, Input, Chip, Badge, Card, Avatar. Composed into GearCard, ListRow, StatCard, ActionItem, ShortcutGrid. All with Inter font via expo-font.
